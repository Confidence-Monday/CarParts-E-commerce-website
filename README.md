# Car-part-e-commerce-website
End-to-End Functional and UX Testing, to validate the core business workflows of the car parts e-commerce platform, including authentication, product browsing, cart management, checkout, payment process, and contact functionality.

**Environment**: Web Application

**Browser**: Chrome 145

**Windows:** x86

**Test Duration**: 4 Days (Feb 16 - Feb 20)

**Test Data:** Sample user accounts and test payment data


**Scope**

     In Scope:

User Registration and Login

Product Browsing and Navigation

Cart and Checkout Flow

Address Management

Contact Module

Payment Flow (partial due to blocking defects)
